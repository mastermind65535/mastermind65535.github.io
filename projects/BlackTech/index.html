BlackTech
Latest Version: 4.1.0
2024-11/08

Project Overview

BlackTech project is a DLL (Dynamic Link Library) malware developed in C++, C#, and Python using AI (ChatGPT, Copilot, etc). It collects most of the information available to the system and extracts the Chrome browser's password list as a key feature and sends it to an external SMTP (Gmail) server.

As the project is registered on VirusTotal, its patterns and hashes may be detected by most antivirus software in a short period of time.
