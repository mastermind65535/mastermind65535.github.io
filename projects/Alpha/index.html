Project Overview

Alpha project is a RAT (Remote Access Trojan) that exploits a reverse shell connection. The project divides large numbers of clients into sessions and supports secret communications through AES encryption (which was supported in future versions but is now removed).

Due to the data removal of AES encryption system, the project level will be downgraded. Advanced → Intermediate
