Purposes

This project is a Proof of Concept (PoC) malware designed to evaluate the ability of ransomware detection of anti-virus software and assess their effectiveness in responding to and minimizing the impact of already executed malicious code. (No malicious actions are included in the source code.)

Usage

Execution

You must run server.py on the same server or it will not be activated.
SharpCry.exe --debug &:: DEBUG MODE: PRINTS ALL INFORMATION
SharpCry.exe --force &:: ACTION MODE: ENCRYPT ALL FILES

Configuration

public static string __KILL_SWITCH = "KILL_SWITCH_SERVER"; // i.e. http://localhost:8080/

The kill switch will be automatically activate if the program cannot "access" to the link. This means if your kill switch server is NOT active, it will shutdown.

Manual Compilation

  1. Load the SharpCry.sln in Visual Studio 2022.
  2. Publish the solution.
  3. Go to bin/Release/net8.0/win-x64/publish/win-x64/
  4. Use the SharpCry.exe.

    Prebuilt Executable

    !WARNING! Prebuilt executable does not support remote server. This mean the program ONLY works on your device.
